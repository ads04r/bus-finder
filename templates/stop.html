Get the Data!

We encourage re-use of data. To this end we provide machine readable versions of this page in various formats.

  • Turtle - Turtle is a representation of RDF, the standard format for linked open data.
  • RDF/XML - RDF, represented in XML.
  • JSON - JavaScript Object Notation, useful when writing web applications.

If you prefer, you can visit the University of Southampton's Data Site to access the raw data used in the building of this site. Here you will find many more datasets in the same format, and a SPARQL endpoint for querying the data from the web.

Finally, if you produce something cool using the data available from this site, please let us know!
